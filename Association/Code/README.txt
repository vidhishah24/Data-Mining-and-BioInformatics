Steps to run the code:

1. Open Command Prompt and go to the ".jar" file directory.
2. Place the data file "associationruletestdata.txt" in the same folder as the jar file.
3. Type the following command: java -jar Apriori.jar
4. The program will ask for user input for Support and Confidence value.
5. It will display size of frequent itemsets for all lengths and generate the rules.
6. Input the query that needs to be executed on the rules generated.
7. Enter next query or EXIT to stop the program.
8. You can go through the source code Apriori.java
